# Kavinya Karthikeyan Portfolio

A personal portfolio website built with React, Vite, Tailwind CSS, and Framer Motion. This website presents my profile, education, technical skills, internships, projects, and contact details in a modern and responsive interface.

## Live Demo

[View Portfolio](https://kavinya140.github.io/Kavinya-Karthikeyan-Portfolio/)

## GitHub Repository

[View Source Code](https://github.com/kavinya140/Kavinya-Karthikeyan-Portfolio)

## Features

- Responsive portfolio design
- Animated hero section and smooth transitions
- About, skills, projects, and journey sections
- Contact section with direct links to email, phone, LinkedIn, GitHub, and LeetCode
- Dark and light theme toggle
- Built with reusable React components

## Tech Stack

- React
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React

## Getting Started

### Install dependencies

```bash
npm install
```

### Run the project locally

```bash
npm run dev
```

### Build for production

```bash
npm run build
```

## Deployment

This portfolio is deployed using GitHub Pages through GitHub Actions.
